Welcome to velocyto.py!
====================================

Velocyto is a library for the analysis of RNA velocity.

Velocyto includes a :ref:`command line tool <cli>` and an :ref:`analysis pipeline <analysis>`

.. warning::
    **CLI** API changed from version 0.11.0. This breaks most previous commands, please refer to the updated :ref:`CLI tutorial <cli>` and :ref:`API session <cliapi>` to know more about the changes.

.. warning::
   This documentation page refers to an alpha version (|version|) of the software. API will be subject to changes until 1.0 release. Please report issues on our `Github page <https://github.com/velocyto-team/velocyto.py>`_.
